# 🚀 Форк zapret для личного использования

Этот проект является форком [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) и предназначен для личного использования.

## Отличия от оригинального проекта: 🔧

-   По умолчанию включен **Game Filter** для блокировки игровых серверов.
-   Обновлен **ipset** для улучшения производительности и совместимости.
-   Добавлены дополнительные стратегии обхода.
-   Добавлены списки **custom-include**, **custom-exclude**, **custom-ipset-include** и **custom-ipset-exclude** для упрощения обновления. Рекомендуется добавлять свои списки именно в эти файлы для удобства управления и обновления.
